JOB SCAM ALERT SYSTEM
Overview

The Job Scam Alert System is a web-based application designed to identify and alert users about fraudulent job postings using Machine Learning and web technologies. The system analyzes job descriptions, suspicious keywords, company details, and other factors to classify whether a job posting is genuine or potentially fraudulent.

This project helps job seekers stay safe from fake job offers and online recruitment scams.

Problem Statement

Online job scams are increasing rapidly and many job seekers become victims of fraudulent job offers. This project aims to provide a smart solution that automatically detects suspicious job postings and alerts users before they apply.

Objectives
Detect fake job advertisements

Step 2: Navigate to Project Directory
